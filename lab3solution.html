Solution 1: Arithmetic Operations

// C program for arithmetic operations
#include <stdio.h>

int main() {
    int num1, num2;
    printf("Enter first integer: ");
    scanf("%d", &num1);
    printf("Enter second integer: ");
    scanf("%d", &num2);

    printf("Sum: %d\n", num1 + num2);
    printf("Difference: %d\n", num1 - num2);
    printf("Product: %d\n", num1 * num2);

    return 0;
}

Explanation

This program performs basic arithmetic. It starts by including the stdio.h header for standard input/output functions. Inside the main function, two integer variables, num1 and num2, are declared. The user is prompted to enter two numbers, which are read using scanf and stored in these variables. Finally, it calculates their sum, difference, and product directly within printf statements and displays the results.

Solution 2: Area of a Square

// C program for area of a square
#include <stdio.h>

int main() {
    int side;
    printf("Enter the side of the square: ");
    scanf("%d", &side);

    int area = side * side;
    printf("Area of the square: %d\n", area);

    return 0;
}

Explanation

This code calculates the area of a square. It prompts the user for the length of one side and stores it in the integer variable side. It then calculates the area by multiplying the side by itself (side * side) and stores the result in the area variable. The final calculated area is then printed to the console.

Solution 3: Area of a Triangle

// C program for area of a triangle
#include <stdio.h>

int main() {
    float base, height;
    printf("Enter the base of the triangle: ");
    scanf("%f", &base);
    printf("Enter the height of the triangle: ");
    scanf("%f", &height);

    float area = 0.5 * base * height;
    printf("Area of the triangle: %.2f\n", area);

    return 0;
}

Explanation

This program calculates the area of a triangle. Since the base, height, or area can be decimal numbers, we use the float data type. After getting the base and height from the user, it applies the formula Area = 0.5 * base * height. The result is printed using the %.2f format specifier, which displays the number with exactly two decimal places.

Solution 4: Fahrenheit to Celsius

// C program to convert Fahrenheit to Celsius
#include <stdio.h>

int main() {
    float fahrenheit;
    printf("Enter temperature in Fahrenheit: ");
    scanf("%f", &fahrenheit);

    float celsius = (fahrenheit - 32) * 5.0 / 9.0;
    printf("Temperature in Celsius: %.2f\n", celsius);

    return 0;
}

Explanation

This program converts a temperature from Fahrenheit to Celsius. It reads a temperature in Fahrenheit as a float. The conversion formula (°F - 32) * 5/9 is then applied. It's important to use 5.0 and 9.0 to ensure floating-point division; otherwise, integer division would result in 5/9 = 0, giving an incorrect result. The final Celsius temperature is printed, formatted to two decimal places.

Solution 5: Area of a Circle

// C program for area of a circle
#include <stdio.h>
#define PI 3.14159

int main() {
    float radius;
    printf("Enter the radius of the circle: ");
    scanf("%f", &radius);

    float area = PI * radius * radius;
    printf("Area of the circle: %.2f\n", area);

    return 0;
}

Explanation

This program finds the area of a circle. We first define a constant PI using the #define preprocessor directive for better readability. The program then reads the circle's radius from the user. Using the formula Area = PI * r², it calculates the area and stores it in a float variable. The result is then printed to two decimal places.
